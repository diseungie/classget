スマホにインストールする方法

iPhoneユーザー

  1. Safariでクラゲ百科事典を開く
  2. 下の真ん中のボタンをクリック
  3. ホーム画面に追加をタップ

Androidユーザー

Chrome
  1. Chromeでクラゲ百科事典を開く
  2. 右下のボタン...をタップ
  3. ホーム画面に追加をタップ
Androidブラウザー
  1. Androidブラウザーでクラゲ百科事典を開く
  2. 右上のブックマークボタンをタップ、ブックマークをする
  3. ブックマークボタンを長押し
  4. ショートカットを作成をタップ
